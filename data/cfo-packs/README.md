# CFO data packs (drop zone)

Place new QuickBooks / CFO `.xlsx`, `.xls`, or `.csv` exports here.

Then run:

```bash
npm run build:cfo-packs
```

The KPI Reporting Hub loads every pack automatically under **CFO / QuickBooks Packs**.

Also scanned:

- `external_docs/exports/qbo/`
- `external_docs/exports/cfo/`
