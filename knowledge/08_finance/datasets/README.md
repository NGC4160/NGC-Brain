## Reading rules

1. **QBO** = cash-basis financial performance  
2. **HCP completed jobs** = production for the month  
3. **HCP paid-in-full** = collections (includes prior-period carryover)  
4. **Staff attribution** is directional — shared jobs double-count; use unique company totals for production  
5. **Do not quote** customer PII, bank account numbers, or vendor payment credentials from source PDFs
