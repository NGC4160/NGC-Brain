# SOP-29 — Documentation & Photo Standard

**Owner:** All techs + Christine  
**Last verified:** 2026-07-12  
**Status:** Draft — ready for review

## Purpose

Transfer-ready documentation so any tech or second location can understand the job.

## Minimum

- Before photos (complaint area + intake set)
- After photos for visible repairs / conversions
- Fault codes / .CPF when electric diag used
- Parts used list on job
- Customer-facing summary of work performed

## Rules

- Attach to HCP customer/job — not only personal phone.
- No passwords or unnecessary PII in notes.
- Lithium: include pack SKU / voltage in notes.

## Done when

Next person can estimate, repair, or defend the job from the file alone.
