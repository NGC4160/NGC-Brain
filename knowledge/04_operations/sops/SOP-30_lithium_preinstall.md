# SOP-30 — Lithium Pre-Install Checklist

**Owner:** Assigned technician + Service Manager  
**Last verified:** 2026-07-12  
**Status:** Draft — ready for review

## Purpose

Do not start a Professional Kit install until money and parts are verified.

## Checklist (all required)

- [ ] Deposit received $1,800 (SOP-05) — or SM stock exception noted
- [ ] Correct SKU on hand: 36 / 48 / MINI / 150AH / 72V
- [ ] Lithium-specific charger present
- [ ] DC-DC converter present
- [ ] Battery monitor / SOC meter present
- [ ] Cart voltage matches kit
- [ ] Board card [Li]; day tracker started (SOP-32)
- [ ] Bay time available (~6 hours)

## Fail

Any unchecked box → stay PARTS or INTAKE; do not fake IN REPAIR.
