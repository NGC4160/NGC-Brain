# SOP-31 — Professional Kit Install (36/48/MINI/150AH/72V)

**Owner:** Assigned technician  
**Last verified:** 2026-07-12  
**Status:** Draft — ready for review

## Purpose

Install Professional LiFePO4 kits consistently (~6 hours typical).

## High-level sequence

1. Pass SOP-30 pre-install checklist.
2. Photo lead-acid compartment; disconnect safely.
3. Remove lead-acid set; stage cores (SOP-35).
4. Mount lithium pack per kit instructions for voltage/SKU.
5. Install lithium charger, DC-DC (36/48→12V 20A), SOC monitor.
6. Torque/lugs/cables; fuse protection as specified.
7. Power-up checks; programmer/monitor as required.
8. Add-ons only if sold (SOP-34).
9. QC + test drive (SOP-28 / SOP-33).

## Rules

- Professional Kits only — no NGC Conversion.
- Voltage-specific quirks: confirm MINI fitment; 72V and 150AH per kit docs.
- Change order if extra repair found mid-install (SOP-25).

## Done when

Cart passes QC; tracker install + QC boxes checked.
