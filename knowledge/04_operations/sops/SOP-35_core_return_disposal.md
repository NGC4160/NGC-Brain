# SOP-35 — Core Return & Disposal

**Owner:** Tech + Christine (fees)  
**Last verified:** 2026-07-12  
**Status:** Draft — ready for review

## Purpose

Handle lead-acid cores and fees correctly on battery R&R and lithium conversions.

## Fees

- **$20** per 6V or 8V battery
- **$30** per 12V battery

## Steps

1. Count batteries removed; note on job.
2. Apply core charges on invoice unless cores returned/kept per policy.
3. Store cores safely for scrap/return vendor (SOP-61 handling).
4. Do not leave acid batteries in customer area or unmarked piles.
