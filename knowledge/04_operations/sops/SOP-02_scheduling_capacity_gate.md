# SOP-02 — Scheduling & Capacity Gate

**Owner:** Office Coordinator (Christine)  
**Backup / override:** Service Manager (Ryan)  
**Last verified:** 2026-07-12  
**Applies to:** All NGC locations  
**Status:** Draft — ready for review

## Purpose

Book drop-offs and lithium starts only when the shop has capacity — so the 2–3 day lithium promise and repair turnaround stay real.

## When to use

Before confirming any drop-off date, lithium start, or converting a waitlist job to a scheduled arrival.

## Prerequisites

- Know current WIP stoplight from board / morning huddle (SOP-10 / SOP-11)
- Deposit status for parts-dependent jobs (SOP-05)
- Customer has approved diagnostic terms or estimate when required (SOP-01 / 03 / 04)

## Hard rules

| Do | Do not |
|----|--------|
| Check stoplight before booking | Promise a bay when WIP is red |
| Prefer waitlist when full | Overbook “just this once” without SM |
| Confirm deposit path for Li/batteries | Schedule lithium install before deposit plan is clear |

## Capacity gates

| Stoplight (WIP lanes 2–5) | Booking rule |
|---------------------------|--------------|
| **Green (0–4)** | Book normally |
| **Yellow (5–6)** | Repairs OK; **new lithium** only if kit in stock + SM OK |
| **Red (7+)** | **No new drop-offs** — waitlist only until finish list clears (SOP-12) |

Other caps: lithium in progress max **4**; needs scheduling desk max **5**.

## Steps

1. Customer ready to schedule → check board header / ask Ryan if unsure.
2. If red → place on waitlist; give no firm start date.
3. If yellow + lithium → confirm with Ryan before booking.
4. If green → book drop-off or Roy pickup; add HCP appointment; create/update INTAKE card when cart is imminent.
5. Lithium: remind **$1,800 deposit** before kit order (SOP-04 / 05).
6. Job note: `Scheduled YYYY-MM-DD — stoplight [G/Y/R]`.

## Done when

Appointment matches capacity rules; customer knows drop-off vs pickup; waitlisted jobs are not fake-scheduled.

## Exceptions

Emergency / comeback / warranty — SM only may bump the gate; document on job.
