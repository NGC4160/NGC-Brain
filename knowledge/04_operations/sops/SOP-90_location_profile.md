## Purpose

One page of **local variables** per shop so Core SOPs stay company-wide.

## Procedure

1. Copy `knowledge/01_company/location_profile_template.md` to `knowledge/01_company/locations/<slug>_profile.md`.
2. Fill address, phone, hours, tax, P&D zones, vendors, roster, WIP soft-open caps.
3. Point staff to that file for local fees/hours — not ad-hoc texts.
4. Covington baseline already sketched in the template footer.
