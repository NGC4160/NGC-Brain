# SOP-04 — Quote & Estimate (Professional Lithium)

**Owner:** Office Coordinator (Christine) — sends quotes once created  
**Backup / creates complex quotes:** Service Manager (Ryan)  
**Last verified:** 2026-07-12  
**Applies to:** All NGC locations (prices from company pricebook)  
**Status:** Draft — ready for Christine pilot

## Purpose

Give every lithium lead the same accurate Professional Kit quote — correct SKU, deposit, turnaround, warranty, pickup/delivery, and optional add-ons — with no discontinued products and no over-promises.

## When to use

Customer wants a lithium conversion (or “switch to lithium,” “lithium batteries,” “get rid of lead-acid”) by phone, text, form, or walk-in.

If the cart also needs unknown electrical diagnosis before conversion, run diagnostic path (SOP-01 / SOP-03) first or in parallel as Service Manager directs.

## Prerequisites

- Intake photos when possible (SOP-01): front of cart, battery compartment, data tags — to confirm make/model/voltage
- Current Professional Kit prices from [lithium_conversions.md](../../02_products/lithium_conversions.md) / HCP pricebook
- Location known for P&D fee (SOP-01)
- Shop capacity check before promising a start date (SOP-02 / [shop_throughput.md](../shop_throughput.md))

## Hard rules (non-negotiable)

| Do | Do not |
|----|--------|
| Quote **Professional Kit** SKUs only | Quote **NGC Conversion**, TEST PARTIAL KIT, or old lower price points |
| State **$1,800 deposit** before kit order | Promise “we’ll order now, pay later” |
| Say **2–3 business days** typical after parts/deposit (sometimes same day if kit in stock + bay open) | Guarantee same-day without Service Manager OK |
| Quote add-ons **separately** | Bundle charger port / reducer / fuse box into base kit price |
| Confirm voltage with customer / photos / SM | Guess 36 vs 48 vs 72 |

**Customer-supplied lithium install ($549):** do **not** quote unless Service Manager approves that job.

## Steps

### 1. Confirm cart & voltage

1. Make/model/year if known.
2. Current system voltage (36 / 48 / 72) from customer or data-tag photos.
3. If voltage unclear → request photos (SOP-01) or park for Service Manager before locking a SKU.
4. MINI vs standard 48V: ask if compartment is tight / carryall / limited space — or let SM choose after photos.

### 2. Capacity gate (before promising dates)

- If lithium WIP is at limit or Service Manager has paused lithium booking → quote price/deposit/warranty, but **do not** promise a start date. Place on waitlist.
- Same-day language only if SM confirms kit in stock **and** bay open.

### 3. Build the quote (who creates)

| Situation | Who creates estimate in HCP |
|-----------|----------------------------|
| Straightforward Professional Kit + clear voltage + optional add-ons | Service Manager creates; **Office Coordinator sends** (per SOP-01) — or OC creates if SM has authorized that SKU pattern |
| Unclear voltage, damage, controller issues, customer-supplied pack, fleet | **Service Manager creates** |

Office Coordinator sends all estimates **except** those that still need Service Manager to create them first (SOP-01).

### 4. What every lithium quote must include

Checklist before sending:

- [ ] Correct Professional Kit SKU and price  
- [ ] Tax note (plus tax)  
- [ ] **$1,800 deposit** — parts ordered after deposit posts (SOP-05)  
- [ ] Typical **2–3 business day** turnaround after deposit/parts (not a guarantee of same day)  
- [ ] **5-year** battery + BMS warranty  
- [ ] What’s included (charger, DC-DC, monitor, disposal, install)  
- [ ] Core charges if lead-acid coming off  
- [ ] P&D fee for their location  
- [ ] Optional add-ons listed separately (or “ask us about…”)  
- [ ] Shop-only — no mobile service  

### 5. Send & follow up

1. Send estimate via HCP (text/email).
2. Job note: `Lithium quote sent YYYY-MM-DD — [SKU] — dep $1800`.
3. If approved → run **SOP-05** (collect deposit) before any kit order.
4. If no response: one follow-up after 24–48 hours; then stop chasing (same spirit as SOP-01 lead follow-up unless customer re-engages).

### 6. After approval

1. Deposit collected (SOP-05) → order release to Service Manager.  
2. Lithium day tracker starts (SOP-32 / shop throughput).  
3. At delivery/pickup: care guide (lithium charger only, SOC guidance) — SOP-33.

## Scripts

**Phone — package intro**

> We install Professional lithium conversion kits in the shop. For a [36/48/72]V cart, the Professional Kit is $[price] plus tax. That includes the lithium pack, lithium charger, DC-DC for accessories, battery monitor, disposal of your old batteries, and installation. Warranty is five years on the battery and BMS.

**Deposit & timing**

> We need an $1,800 deposit before we order the kit. After the kit is here, install is typically about two to three business days — sometimes same day if we have the kit and a bay open.

**Add-ons**

> A lot of customers add a charger port, a voltage reducer, or a fuse box for accessories. Those are separate line items if you want them.

**Do not say**

> We still sell the older NGC Conversion kit / cheaper lithium package.  
> We’ll order it today and you can deposit whenever.  
> We come to you / mobile install.  
> Guaranteed done tomorrow.  
> Customer-supplied battery install is $549 (unless Ryan approved).

**Text / estimate summary (template)**

> Hi [Name] — here’s your Neighborhood Golf Carts lithium quote:  
> • [36V/48V/MINI/150AH/72V] Professional Kit installed — $[price] + tax  
> • Includes lithium pack, lithium charger, DC-DC, monitor, old battery disposal, install  
> • Warranty: 5 years battery + BMS  
> • Deposit to order: $1,800 (balance when ready)  
> • Typical turnaround after parts: 2–3 business days  
> • Pickup/delivery: [free / $99] for your area  
> Optional: charger port, accessory reducer, fuse box — ask if you want those added.  
> Reply here or call 985-402-1206 with questions.

## Done when (exit criteria)

| Path | Done when |
|------|-----------|
| Quote sent | Correct SKU/price/deposit/SLA/P&D in estimate; job noted; customer has link |
| Quote approved | Handoff to SOP-05 deposit collection |
| Cannot quote yet | Photos/voltage/SM review requested; customer told next step; not left hanging without a reason |
| Declined / no reply | Noted in HCP; not on active lithium WIP |

## Exceptions

| Situation | Action |
|-----------|--------|
| Wrong voltage quoted | Void/revise immediately; SM reviews before re-send |
| Cart needs repair before conversion | Quote diagnostic/repair first or combined plan per SM — do not hide issues |
| Pricebook shows NGC Conversion SKU | **Do not use** — Professional Kit only; flag for pricebook cleanup |
| Customer haggles below list | SM only — OC does not discount kits |
| Fleet / multi-cart | SM creates package terms; OC sends |
