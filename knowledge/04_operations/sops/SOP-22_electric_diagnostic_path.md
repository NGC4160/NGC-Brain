# SOP-22 — Standard Electric Diagnostic Path

**Owner:** Assigned technician  
**Escalation:** Peyton per SOP-27  
**Last verified:** 2026-07-12  
**Status:** Draft — ready for review

## Purpose

Shop-only standard path for electric cart diagnostics (replaces mobile-oriented legacy Drive SOPs).

## Sequence

1. Static voltage + IR check; visual inspection
2. DVOM setup for load test
3. Handheld programmer to motor controller
4. Review/clear fault codes (photo or save .CPF)
5. Monitor review
6. Log file for complaint
7. Test drive with voltage monitoring
8. Save post-drive .CPF
9. Document findings → SOP-24 estimate handoff

## Rules

- Diagnostic fee $179 path already sold via SOP-03 — still document thoroughly.
- Intermittent / beyond handheld → SOP-27.
- No parts order from diag without deposit when required (SOP-05).

## Done when

Findings in HCP; codes/files saved; handoff to estimate or repair.
