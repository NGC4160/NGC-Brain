# SOP-34 — Lithium Add-On Installs

**Owner:** Assigned technician  
**Quoted by:** Christine/Ryan (SOP-04)  
**Last verified:** 2026-07-12  
**Status:** Draft — ready for review

## Purpose

Install sold add-ons only — not freebies assumed into the kit.

## Common add-ons (confirm pricebook)

| Add-on | Approx. |
|--------|--------:|
| Charger port w/ extension | $199–$269 |
| 20A accessory voltage reducer | $225 |
| 12V accessory fuse box | $139–$149 |

## Steps

1. Confirm line item on approved estimate.
2. Install per parts instructions during or after kit install.
3. Document on job; photo if visible customer feature.
4. Function-check with QC.
