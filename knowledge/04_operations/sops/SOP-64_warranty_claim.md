# SOP-64 — Warranty Claim (Battery / BMS / Parts)

**Owner:** Service Manager (Ryan)  
**Intake:** Christine  
**Last verified:** 2026-07-12  
**Status:** Draft — ready for review

## Purpose

Process warranty claims with proof — especially 5-year lithium battery/BMS.

## Steps

1. Gather prior invoice, install date, photos, symptom.
2. Determine: lithium 5yr battery/BMS vs lead-acid vendor warranty vs workmanship.
3. Ryan approves claim path with vendor if needed.
4. Do not auto-waive diagnostic if cause is unrelated — SM decides.
5. Document outcome on job; update customer (SOP-06).
