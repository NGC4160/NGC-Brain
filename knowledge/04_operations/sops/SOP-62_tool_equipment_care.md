# SOP-62 — Tool / Equipment Care

**Owner:** Technicians  
**Last verified:** 2026-07-12  
**Status:** Draft — ready for review

## Purpose

Keep diagnostic tools working — DVOM, programmers, chargers.

## Rules

- Return programmers/DVOM to charged dock/cabinet daily
- Do not use lithium chargers as general power supplies
- Report broken tools to Ryan same day
- Keep firmware cables labeled
