# SOP-43 — Vehicle & Trailer Safety

**Owner:** Driver (Roy)  
**Last verified:** 2026-07-12  
**Status:** Draft — ready for review

## Purpose

Prevent load-shift and road incidents on every P&D trip.

## Checklist

- [ ] Trailer lights, tires, coupler, safety chains
- [ ] Cart centered; wheel chocks / straps secure
- [ ] Key off; brake set if applicable
- [ ] Ramps clear; no bystanders in path
- [ ] Gross weight within limits
- [ ] Secure loose chargers/parts in cab/bin

## Rule

If it doesn’t feel safe, don’t tow — call Ryan/Christine.
