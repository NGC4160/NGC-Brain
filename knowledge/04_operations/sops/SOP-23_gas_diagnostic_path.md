# SOP-23 — Gas Cart Diagnostic Path

**Owner:** Assigned technician  
**Last verified:** 2026-07-12  
**Status:** Draft — ready for review

## Purpose

Diagnose gas carts with the same documentation discipline as electric.

## Sequence (shop)

1. Visual: leaks, belts, intake, exhaust, battery/starting battery
2. Confirm complaint (no start, dies under load, poor run)
3. Fuel / spark / compression basics per model
4. Test drive when safe
5. Photos + findings → SOP-24
6. 7-point safety (SOP-21)

## Rules

- $179 diagnostic rules still apply (SOP-03).
- No mobile/on-site.
- Escalate unfamiliar EFI/governor issues to SM.
