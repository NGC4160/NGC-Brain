# SOP-26 — Lead-Acid Battery Replacement

**Owner:** Assigned technician  
**Deposit:** Christine SOP-05 ($800 typical)  
**Last verified:** 2026-07-12  
**Status:** Draft — ready for review

## Purpose

Replace lead-acid sets safely with cores handled and warranty notes captured.

## Steps

1. Confirm deposit posted and correct set ordered/on hand (6V/8V/12V).
2. Photo old set / compartment before (SOP-29).
3. Install per manufacturer practice; torque/cables clean.
4. Cores: SOP-35 — core fees $20 (6V/8V) or $30 (12V).
5. Note warranty terms on job (e.g. Crown 18-mo where applicable).
6. QC + water levels / connections (SOP-21 / 28).

## Done when

Cart runs, cores processed, ready for customer.
