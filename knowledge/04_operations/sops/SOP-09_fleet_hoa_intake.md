# SOP-09 — Fleet / HOA Intake

**Owner:** Office Coordinator + Service Manager  
**Last verified:** 2026-07-12  
**Status:** Draft — ready for review

## Purpose

Intake multi-cart commercial/HOA work with clear billing vs site contacts.

## Steps

1. Capture **billing contact** and **site contact** separately.
2. List carts (make/model/issue) by unit ID — not one vague job if multiple carts.
3. SM sets deposit/PO terms before parts (may differ from retail $1800/$800).
4. Peyton trigger if pattern across fleet (SOP-27).
5. Schedule in batches that respect WIP (SOP-02 / 12).

## Done when

Each cart trackable; money terms written; site knows arrival windows.
