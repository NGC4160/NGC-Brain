# SOP-51 — Bay Parts Pull & Return

**Owner:** Technicians  
**Last verified:** 2026-07-12  
**Status:** Draft — ready for review

## Purpose

Reduce shrink and “mystery missing parts.”

## Rules

- Pull only for your assigned invoice #.
- Return unused parts to labeled location same day.
- Do not borrow from another job’s labeled kit without Ryan.
- Record parts used on job (SOP-29).
