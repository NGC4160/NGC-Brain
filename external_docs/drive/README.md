# Google Drive mirror (local sync)

Files here are pulled from Google Drive by:

```bash
./scripts/sync/run_google_drive_sync.sh
```

Configure what syncs in `config/google_drive_sync.json`.

Sync manifest: `external_docs/exports/drive/sync_manifest.json`

This folder is gitignored (can be large). Brand logos also copy to `external_docs/assets/`.
