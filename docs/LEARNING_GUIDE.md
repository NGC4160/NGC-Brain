# NGC Dashboard — Learning Guide

Interactive walkthrough built into the Operations Dashboard so Ryan, Christine, and staff can learn every module by doing.

## How to open it

1. Sign in at https://ngc4160.github.io/NGC-Brain/
2. On first login, tap **Start guide** on the welcome card
3. Or open **Learning Guide** in the sidebar / More menu (`#/guide`)
4. Or tap the green **Guide** button (bottom-right)

## What it covers

| Chapter | What you learn |
|---------|----------------|
| Start here | Dashboard purpose + navigation (sidebar / bottom tabs / More) |
| Home Dashboard | Date range, HCP sync, KPI cards, revenue + active jobs |
| Customer Intake | Lead capture SOP flow |
| SOPs | Role-filtered procedure library |
| KPI Hub | Targets, ranges, CSV/PDF export |
| Manuals & Files | Pin manuals for the shop floor |
| Jobs | Filters, create/assign, tech assignment view |
| Status Board | WIP columns vs shop target |
| QC Form | Tech completion checklist |
| Agent Input | Mid-job logging |
| Invoicing | AR + deposit gates |
| Settings | Staff passcodes, Drive manuals, HCP/QBO |
| Roadmap | Inventory / CRM / Scheduling (coming soon) |

Steps you cannot access for your role are skipped automatically.

## Controls

- **Next / Back** — move through the tour (also ← → / Enter / Esc)
- **Resume** — continue from last step (saved on this device)
- **Restart** — clear progress and start over
- Jump to any step from the Learning Guide page checklist

Progress is stored in `localStorage` as `ngc-learning-guide-v1` (this browser only).
