# Instructor Master Checklist — Full 10-Week Cohort

Use this as the bay/ops spine. Detail lives in each `weeks/week_XX_*.md` prep section.

## Before Week 1
- [ ] Confirm class roster size (ideal 6–8; max 10 without assistant)
- [ ] Reserve 4–6 carts (36/48V LA mix, ≥1 lithium, 1–2 gas)
- [ ] Inventory PPE, DMMs, load tester, hydrometers, extinguisher, eyewash, spill kit
- [ ] Obtain OEM manuals / diagram access for every training cart
- [ ] Print student binders: handouts + labs index
- [ ] Post safety poster; mark road-test loop
- [ ] Review Program Guide grading weights & critical items

## Weekly rhythm (every week)
- [ ] Day-before: complete that week’s Instructor Preparation Guide  
- [ ] Print quizzes + lab sheets  
- [ ] Stage planted faults; log secret key for reset  
- [ ] Opening safety 2–5 min  
- [ ] Collect worksheets; update competency tracker  
- [ ] Reset carts to safe condition before fleet use

## Before Week 10 finals
- [ ] Practical fault bank ready (unique per student if possible)
- [ ] Exams printed; answer key secured
- [ ] Certificates pre-filled except scores
- [ ] Makeup labs completed

## After cohort
- [ ] Score & issue certificates  
- [ ] Reset all faults  
- [ ] Write after-action in `ITERATION_LOG.md`  
- [ ] Tool/PPE damage inventory
