# Student Handout — Wiring Symbol Glossary (Golf Cart Focus)

Keep this sheet in your binder. Always confirm symbols against the OEM legend for the diagram in use.

| Symbol / representation | Meaning | Cart context |
|---|---|---|
| Long/short parallel lines | Battery / cell | Pack shown as multi-battery string |
| Circle with M / motor icon | Motor | Traction motor |
| Coil rectangle / zigzag + contacts | Solenoid / contactor / relay | Main solenoid, F&R contactors |
| Fuse zig-zag or bar | Fuse | Control or accessory protection |
| Switch (SPST/SPDT styles) | Switch | Key, direction, microswitches |
| Potentiometer / resistor with arrow | Variable resistor | Older throttle pots |
| Diode triangle/bar | Diode | Flyback, lighting, polarity protection |
| LED | Indicator | Status / charge lamps |
| Connector numbered pins | Connector | Harness plugs — pin IDs matter |
| Ground symbol | Chassis/battery ground | Bad grounds = ghost faults |
| Heavy line vs light line | Power vs signal (convention varies) | Prefer OEM wire gauge callouts |

## Color code caution
Wire colors are **not universal** across Club Car, EZ-GO, and Yamaha. Treat color as a hint only after the correct model-year diagram is open.

## Quick tracing rules
1. Match diagram to cart tag (make/model/year).  
2. Trace power path first on paper.  
3. Continuity with power removed; voltage drop under controlled load for high current cables.  
4. Never copper-slug a fuse.
