## Quick Start for Instructors

1. Read **Program Guide** end-to-end once before Week 1.
2. Print/copy **labs/** and **handouts/** for the student binder.
3. Each week: follow that week’s file — Session 1 (theory), Session 2 (lab).
4. Use **Instructor Preparation** section at the top of each week file *the day before*.
5. Week 10: run final practical + written exam from `final_assessment/`.

## Safety Emphasis (Non-Negotiable)

Post and enforce Appendix-style shop rules from handouts. Every session opens with a 2–5 minute safety reminder. Treat every pack as energized until verified zero energy with a meter.
