# Student Handout — Preventive Maintenance Checklists

Customize intervals to fleet hours and OEM schedules. Lithium tasks differ from flooded lead-acid.

---

## Daily Pre-Use (Operator / Starter Tech)

- [ ] Tire condition & pressure (all four)  
- [ ] Obvious leaks / loose parts / damaged cables  
- [ ] Brakes & parking brake function  
- [ ] Lights / horn if equipped  
- [ ] Unusual noises on short move  
- [ ] Flooded: quick look for low water / wet acid trails  
- [ ] Lithium: charger connected overnight as policy; SOC not critically low  
- [ ] Steering responds normally  

---

## Weekly

- [ ] Clean battery tops (flooded); neutralize corrosion  
- [ ] Check/torque accessible battery connections to spec  
- [ ] Charger cords & receptacles inspection  
- [ ] Wash undersides if coastal/sandy (avoid pressure into bearings/controllers)  
- [ ] Brake adjustment check as applicable  
- [ ] Document hours / issues in log  

---

## Monthly

- [ ] Deeper cable/insulation inspection  
- [ ] Controller area debris / moisture check  
- [ ] Suspension bushings / steering play spot check  
- [ ] Transaxle seepage check  
- [ ] Accessory 12V systems / DC-DC (lithium conversions)  
- [ ] Review repeat fault codes / comebacks  

---

## Annual or 500–1000 Hour (as OEM/fleet policy)

- [ ] Full battery bank load test (flooded) / lithium health report  
- [ ] Controller diagnostics / software if applicable  
- [ ] Motor brush inspection (brushed motors)  
- [ ] Brake lining measurements  
- [ ] Transaxle service / fluid per OEM  
- [ ] Torque audit critical fasteners  
- [ ] Road test & sign-off
