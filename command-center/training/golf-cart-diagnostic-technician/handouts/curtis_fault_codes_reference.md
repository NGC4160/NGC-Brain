# Student Handout — Curtis Fault Code Reference (Training Aid)

**Important:** Flash codes vary by controller model and software. This sheet teaches **method**, not a universal code list. Always use:

1. The code chart in the **OEM/service manual for that controller model**, and/or  
2. Curtis Fault Code Translator: https://faultcodes.curtisinstruments.com/

## How to read LED flash codes (general method)

1. Note controller model number on the label.  
2. Observe LED: count flashes in the sequence pattern used by that model (often sequences of flashes with pauses).  
3. Record exactly what you see (e.g., “2 flashes — pause — 1 flash”).  
4. Look up meaning in the **model-correct** chart.  
5. Confirm with pin/voltage tests — codes point to a subsystem, not automatic parts replacement.

## Families referenced in this program
- Curtis **1204 / 1205 / 1268** series (and related) — common in golf/utility conversions and OEM applications  
- Mention **Alltrax** as alternate family — follow Alltrax documentation

## HPD / SRO reminders
- **HPD (High Pedal Disable):** throttle applied at wrong power-up sequence inhibits drive.  
- Teach operators: pedal up → enable/key → then throttle (confirm OEM).  
- Never permanently bypass for customer return.
